## ðŸŽ¯ Common Tasks

### Running Tests

```bash
# Basic test run
pytest tests/ -v

# With coverage
pytest tests/ -v --cov=neural --cov-report=term --cov-report=html

# Generate full report
python generate_test_coverage_summary.py
```

### Viewing Results

```bash
# Markdown summary
cat TEST_COVERAGE_SUMMARY.md

# HTML report (interactive)
start htmlcov/index.html          # Windows
open htmlcov/index.html           # macOS
xdg-open htmlcov/index.html       # Linux
```

### Selective Testing

```bash
# Specific file
pytest tests/parser/test_parser.py -v

# Specific test
pytest tests/parser/test_parser.py::test_parse_simple_model -v

# By marker
pytest tests/ -v -m unit                    # Only unit tests
pytest tests/ -v -m "not slow"              # Exclude slow tests
pytest tests/ -v -m "not requires_torch"    # Exclude PyTorch tests
```

## ðŸŽ¨ Test Markers

Available markers for test categorization:

| Marker | Description |
|--------|-------------|
| `slow` | Tests that take >1 second |
| `integration` | Integration tests |
| `unit` | Unit tests |
| `requires_gpu` | Tests requiring GPU |
| `requires_torch` | Tests requiring PyTorch |
| `requires_tensorflow` | Tests requiring TensorFlow |
| `requires_onnx` | Tests requiring ONNX |

### Before Committing

1. Run tests: `pytest tests/ -v`
2. Check coverage: `pytest tests/ -v --cov=neural`
3. Run linter: `ruff check .`
4. Format code: `ruff check --fix .`

### CI/CD

- Tests run automatically on push/PR
- Coverage uploaded to Codecov
- Multiple Python versions tested (3.8, 3.11, 3.12)
- Multiple OS tested (Ubuntu, Windows)
